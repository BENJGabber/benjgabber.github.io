Mes Projets & Apprentissages

Pourquoi ce projet, mes projets passés et ce que j'en ai tiré

Pourquoi j'ai choisi ce projet

J'ai choisi de réaliser ce portfolio comme projet principal car il rassemble mes centres d'intérêt: le développement web, la présentation claire de mon parcours et la mise en valeur de mes compétences techniques. Ce site me permet de pratiquer HTML, CSS et JavaScript tout en créant un support professionnel pour mes candidatures et mon réseau.


C'est aussi un terrain d'expérimentation: accessibilité, design responsive, mode clair/sombre, et intégration de contenus dynamiques (articles, expériences, contacts). À chaque mise à jour, j'améliore structure, performance et ergonomie.

Mes projets passés

Online Launchpad

Site web "Launchpad" reproduissant un appareil musical appelé "launchpad" qui aurait permis de créer des sons avec des boutons. Ce projet m'a permis de découvrir les bases du JavaScript tout en testant mes competences déjà acquises en HTML et CSS grâce a mes cours et a un mini site web que j'avais fait en stage de 3ème.

JavaScript HTML CSS

"LDLCLONE"

Création d'un site web clone de la plateforme de vente en ligne "LDLC" avec une interface simple aux couleurs du logo de l'ensemble saint Saint-Luc, j'ai finit par changer de projet car c'etait beaucoup trop ambitieux et je n'avais pas les competences necessaires pour le realiser correctement. J'ai même essayé d'apprendre le react pour le realiser mais je n'avais pas les competences necessaires pour le realiser correctement.

JavaScript HTML CSS

Ce que j'ai appris

  • Organisation d'un site: navigation claire, cohérence visuelle, responsive design.
  • Composants réutilisables: cartes, boutons, formulaires avec styles cohérents.
  • JavaScript côté front: interactions (menu latéral, lightbox, thème clair/sombre).
  • Qualité: lisibilité du code, séparation structure/style/script, test visuel multi-écrans.
